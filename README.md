gitignore.io.sh
===============

A simple [gitignore.io](http://gitignore.io) shell wrapper :

## Usage

    ./gitignore.io.sh [list|template,...]
    
      list : displays a list of all of the currently support gitignore.io templates
      
      e.g. ./gitignore.io.sh c++,linux > .gitignore
